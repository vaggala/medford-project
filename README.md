# CO Adsorption on Pt(111) and Cu(111)

Modeled adsorption of carbon monoxide (CO) on platinum (111) and copper (111) crystal surfaces using Density Functional Theory (DFT) with SPARC and ASE. Computed site-dependent adsorption energies for different binding configurations (e.g., atop and hollow sites) and analyzed how surface structure and metal type influence catalytic behavior.
