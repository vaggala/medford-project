## CO Adsorption on Pt(111) and Cu(111)

Modeled adsorption of carbon monoxide (CO) on platinum (111) and copper (111) surfaces using Density Functional Theory (DFT) with SPARC and ASE.

Computed adsorption energies across multiple binding sites (atop and hollow) and analyzed how surface structure and metal type influence catalytic behavior.
